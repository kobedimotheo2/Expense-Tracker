# Expense Tracker
a desktop app built with java and javaFX that helps you track your daily expenses.
# features 
- add expenses with a description , amount , and category
- view all expenses in a clean,  organized tabble
- see a live running total in Botswana pula (P)
- delete the unwanted expenses with one click
- data is saved automatically, and are saved even after you close the app
- pie chart showing spending brakdown by category
- moden dark them interface

# How to run
 # prerequisites
 - java JDK 8 or higher
 - Maven
 # Steps 
 1. clone the repo
   git clone https://github.com/kobedimotheo2/Expense-Tracker.git

 2. Navigate into the project folder
  
 3. Run the app
    mvn javafx:run
